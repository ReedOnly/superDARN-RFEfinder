Useful commands

https://github.com/ReedOnly/superDARN-RFEfinder.git

Making new local copy
git clone --branch work-in-progress https://github.com/ReedOnly/superDARN-RFEfinder.git


Download from git
git pull origin work-in-progress

Upload to git:
git add . #stages new and modified, without deleted
git add -u #stages modified and deleted, without new

git reset #Undo staging

git commit -m 'comment'
git push origin work-in-progress


Other commands:
git status
git rm file	#Delete file or folder (add -r)

git branch branch_name		#Create new branch
git branch -d branch_name	#Delete local branch
git push origin --delete branch_name	#Delete remote branch
git branch -a 			#List branches
git checkout branch_name	#Change branch


Set up git:
git config --global user.name "ReedOnly"
git config --global user.email "kristian_reed@yahoo.no"


git remote set-url origin https://github.com/ReedOnly/superDARN-RFEfinder.git

git remote -v			#Check current labels

To work on SSL computer:
git config --global http.sslverify false
git remote set-url origin https://ReedOnly@github.com/ReedOnly/superDARN-RFEfinder.git
git config --global --unset-all user.name
git config --global --unset-all user.email

request-pull
git request-pull 0.5-master https://github.com/vtsuperdarn/davitpy.git master
git remote add upstream https://github.com/vtsuperdarn/davitpy.git
git fetch upstream
git merge upstream/master
